# What2Watch

# Description
A simple app that offers suggestions of movies to watch, based on the user's current local weather conditions.

# Technologies Used
- HTML
- CSS
- JavaScript
- jQuery
- jQuery TouchPunch UI
- The Movie Database API
- OpenWeather API
- Geolocation-DB
